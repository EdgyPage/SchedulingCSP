Instructions

How to prepare a roster the Solver will accept — without ever putting employee names or other personal data online.

Why no names?

To protect you and your staff, the Solver only accepts de-identified data. It never sees names, emails, or notes — just anonymous numbers and True/False flags. The uploaded file must have exactly these columns and nothing else:

  • ID Alias — a whole number for each employee: 1, 2, 3, … up to the number of employees, each used once, in any order.
  • One or more Func 1, Func 2, … columns — one per job function, with a True or False in every cell (is this person approved for that function?).

If the file contains a Name column, any other column, blank cells, or IDs that aren't 1…n, the upload is rejected. This is on purpose — it keeps personal data off the internet.

1 Use the roster template (easiest)

The template turns your real roster into an anonymous upload file, and turns the solved schedule back into real names — all on your own computer. Nothing leaves your machine.

⬇ Download the roster template (.xlsx)

Then:

  1. Open it in Excel. On the Encoder tab, replace the example rows with your real employees (their real IDs) and rename the function columns to your real function names, marking each cell TRUE or FALSE.
  2. Click Data → Refresh All (one click). The template assigns each person a random ID Alias and fills the Encoded tab.
  3. On the Encoded tab, select the whole table, copy it into a blank workbook, and Save As a .csv or .xlsx. That is your upload file.
  4. Go to the Solver page, upload that file, set your target headcounts, and Solve.

Don't click Refresh All again after you've uploaded, until you've finished decoding — refreshing picks new random aliases and the old file won't match.

2 Turn the results back into names

After the Solver finishes, download the schedule (.xlsx or .csv). It lists an ID Alias and a Function (like Func 2, or Unassigned) for each person.

  1. Copy the schedule's two columns.
  2. Paste them into the template's Decoder tab.
  3. The real employee identifier and real function name fill in automatically — that's your finished schedule.

Building the file by hand (optional)

If you'd rather not use the template, make a spreadsheet with exactly this shape:

ID AliasFunc 1Func 2Func 3
1TRUEFALSETRUE
2FALSETRUETRUE
3TRUETRUEFALSE
  • The first column must be named exactly ID Alias.
  • Its values must be the whole numbers 1 through n (the number of rows), each used exactly once.
  • Every other column must be named Func 1, Func 2, … (the word Func, a space, then a number).
  • Every cell under a Func column must be True or False — no blanks, numbers, or text.
  • No Name column and no other columns of any kind.

Keep a private copy of which real person each ID Alias refers to, so you can read the results — the template's Decoder tab does this for you.
