Solver

Upload a roster, set a target headcount per task, and generate schedules that fill every task by exactly its target. Leftover staff go to Unassigned. If no exact schedule is possible, you'll get the closest achievable ones — with what's short spelled out — and can download any of them.

1 Upload roster

A de-identified .xlsx or .csv in exactly this shape — no names and no other columns:

ID AliasFunc 1Func 2…Func N
1TrueFalse…True
2FalseTrue…False
3TrueTrue…False

ID Alias = the whole numbers 1…N, each used once. Every Func column holds True/False. The roster template produces this for you — details on the Instructions page.
